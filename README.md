# Spring Boot Project 

## 프로젝트 기획
Spring Boot 를 이용한 쇼핑몰 기능 구현

## 개발 환경

### ★Back End
##### - Spring Boot 2.7.6
##### - intelliJ(IDE)
##### - Java 8, JDK 1.8
##### - ORM : JPA/Hibernate
##### - Build : Gradle

### ★Front End
##### - Bootstrap CSS Framework
##### - Thymeleaf(Template Engine)
##### - jQuery

### ★Database
##### - Postgresql

### ★Server
##### - Tomcat(내장)
